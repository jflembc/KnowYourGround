USDA FSA Coverage

ARC provides revenue-based protection. Payments triggered when county (or farm) revenue falls below 86% of the benchmark revenue.

PLC provides price-based protection. Payments triggered when the effective price falls below the reference price set by Congress. Works best for crops with strong reference prices relative to market.
